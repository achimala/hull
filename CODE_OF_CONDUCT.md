# Code of Conduct

## Our standard

We want this project to be a respectful and welcoming place for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent.
- Give useful feedback on code and ideas.
- Focus on technical issues, not personal attacks.

## Unacceptable behavior

- Harassment, insults, or discriminatory language.
- Trolling, intimidation, or hostile conduct.
- Publishing private information without permission.

## Reporting

If you see a problem, open a private report through repository security reporting,
or contact the maintainers directly.

All reports will be reviewed and handled promptly.
